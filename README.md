Dolby Atmos Sony Xperia 5 V (MTK Port)

📌 About

This is a MediaTek (MTK) port of the Dolby Atmos Sony Xperia 5 V Magisk Module.

The original module was designed for general devices, but it caused:

- Bootloop issues
- RIL / No SIM detection issues

This version fixes those problems and improves compatibility with MTK devices.

---

⚙️ Features

- Dolby Atmos sound enhancement
- Global audio effect support
- Optimized for MediaTek processors
- Improved stability

---

🔧 Changes / Fixes (MTK Port)

- Fixed bootloop issue on MTK devices
- Fixed RIL issue (No SIM detected)
- Improved compatibility with Dimensity chipsets
- Cleaned and optimized module behavior

---

📱 Tested Device

- CMF Phone 1
- Processor: Dimensity 7300

---

⚠️ Requirements

- Root (Magisk / KernelSU / etc.)
- Android 11+
- arm64-v8a architecture
